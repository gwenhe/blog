开始学习 学习方法

简洁至上

以 Markdown 为中心的项目结构，以最少的配置帮助你专注于写作。

Vue驱动

享受 Vue + webpack 的开发体验，在 Markdown 中使用 Vue 组件，同时可以使用 Vue 来开发自定义主题。

高性能

VuePress 为每个页面预渲染生成静态的 HTML，同时在页面被加载的时候，将作为 SPA 运行。

# 吾儿焕友888888

# 像数 1, 2, 3 一样容易

# 在你的项目中安装
yarn add -D vuepress@next

# 新建一个 markdown 文件
echo '# Hello VuePress' > README.md

# 开始写作
yarn vuepress dev

# 构建静态文件
yarn vuepress build
