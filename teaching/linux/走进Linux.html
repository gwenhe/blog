# 什么是Linux

linux是一个操作系统、和我们用的windows、Android、ios一样, 是一个操作系统.

是一种自由和开放源码的类Unix操作系统, 基于POSIX 和 UNIX 的多用户、多任务、支持多线程和多 CPU 的操作系统。

名词解释:

  • 开放源码是什么
    • 简称开源、让源代码开发,可供别人使用.
  • 类Unix
    • 是指和Unix系统相似、Unix是linux的前身
  • 多用户
    • 像windows系统一样的多用户管理
  • 多任务
    • 可以同时运行很多任务
  • 多CPU
    • 可以利用CPU的多个核心

# Linux系统的用途

linux多用于企业的服务器操作系统, 企业的项目都是部署在linux系统的.


讲义:

linux一般呢都是用于企业的服务器操作系统, 企业的项目都是部署在linux系统的. 服务器操作系统你首先知道什么是服务器, 我们用的电脑是服务器吗? 嗯,不是服务器

大家看这张图, 这个企业的机房照片, 里面全部都是服务器 服务器一般都是这样的, 像这样一个竖条一个竖条的这些都是服务器, 这一堆组成一个集群服务器. 而且你看这里有没有显示器? 嗯, 没有显示器 像这些服务都只有主机,而没有显示器, 如果你要想操作这些服务器

而这些服务器一般都在哪, 有钱的公司都是有自己的机房. 大部分一些中小型企业都是租用别人的服务器 像我现在国内比较火的。阿里云 90%的市场份额、腾讯云、华为云、小鸟云、百度云、美团云 而且这种租用是租用的都是云服务器, 什么是云服务器呢, 就是机器都还在阿里的服务器中心, 然后把单台机器租赁给你, 然后只是把服务器的操作权限给到了你.

# Linux 内核及发行版

在上面我们讲的linux系统可以说是linux内核. 而linux发行版是将linux内核与应用程序做了一个打包.

目前市面上较知名的发行版有：Ubuntu、RedHat、CentOS、Debian、SuSE 等。
