# 教学环境

考虑到学员的练习自由性质,以及服务器资源消耗. 在学员电脑预装虚拟机及linux系统、ssh客户端

  • windows虚拟机选择

    • VMware

  • linux发行版的选择

    • 采用使用最为广泛的centos7

  • ssh客户端

    • Xshell6

备注:以防学员误操作,在VMware中做好镜像还原
